Welcome to Dragonfly GameServers 2.0.2!

In order to install Dragonfly GameServers all you have to do is upload the files in their corresponding folders to your webserver.
Then go to your module control panel, in there you will automatically see a selection for GameServers.
Click on "INSTALL" and the module will be installed and ready to go. That's it!
In the event that the installer does not work properly for you please contact a member of our porting team through our site.
We've also provided you with the SQL table file just in case. SQL File

Currently this is the first version of the Dragonfly GameServers module.
If you are an old phpnuke user and want to upgrade your module along with Dragonfly it is not possible at this time.
Simply install the GameServer module and input your data again.
You'll find the new tablesort option is much more friendly to make quick changes to your displayed index.

Remote Installation: (Advanced)
  1. Follow the Standard Installation steps
  2. copy (not move) the modules/GameServers/gameq folder to your remote query server with UDP access
  3. edit remote.php on the remote query server and remove lines 8 and 25 and modify the security code on line 10
  4. login to admin on your main site and click gameservers then configuration
    1. enable gameq remote
    2. enter the full path to your remote query server (ex: http://www.domain.com/gameq)
    3. enter the query code you entered into remote.php

Single Server Block Setup:
Open blocks/block-GameServers-SS.php
on line 9, set $blockserverID to an active servers id#
upload & activate in nuke admin

to use additional blocks just copy block-GameServers-SS.php to a new file
you can call it block-GameServers-SS2.php or whatever else as long as you edit line 3 to match the filename exactly


Core Upgrades (gameq):
This modules uses a modified version of gameq
at its core. upgrading gameq may cause this module not to function
properly but upgrading the core is easy just upload gameq to
/modules/GameServers/gameq
that's it!

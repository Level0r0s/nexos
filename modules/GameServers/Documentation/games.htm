The Supported games will return formatted data. all other games will
return 'Raw' data If you would like to see a template for a game
listed in the admin panel that isn't listed here, send an email to
webmaster@voicesinmyhead.net and i'll put it higher on the list

if a game is not listed in the admin panel, it will be a little
harder to add to this module. the exception is 'spy' games-
(games that use the gamespy protocol for queries) those games are
usually easy to add support for. either way, feel free to send
an email to webmaster@voicesinmyhead.net and i'll see if it can be added.
